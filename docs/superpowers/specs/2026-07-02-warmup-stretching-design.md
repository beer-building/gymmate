# Разминка и растяжка в программах тренировок

Дата: 2026-07-02. Статус: утверждено.

## Задача

Дать возможность добавлять разминку (в начало) и растяжку (в конец) тренировки в программе,
выполнять их по времени в воркауте и получать подсказки растяжек на задействованные мышцы.

## Решение: расширяем exercises, без новой сущности

Разминки/растяжки — те же упражнения (имя, инструкция, мышцы). Отдельная сущность
дублировала бы каталог, join-таблицы программ и логирование — отклонена.

## 1. Схема (PocketBase, миграции в ../gymmate_backend)

- `exercises.kind`: select `strength | warmup | stretching`, required.
  Существующие записи бэкфиллим в `strength`.
- `program_workout_exercises.target_duration_seconds`: number, 0 = не задано.
- `user_program_workout_exercises.target_duration_seconds`: number, 0 = не задано.
- Логи не меняем: в `workout_log_sets` уже есть `duration_seconds`.
  Поле `is_warmup` там — про разогревочные подходы силового упражнения, не смешиваем.

Типы в `src/lib/shared/types/index.ts`: `ExerciseKind`, поле `kind` в `Exercise`,
`target_duration_seconds` в `ProgramWorkoutExercise` и `UserProgramWorkoutExercise`.

## 2. Каталог и exercise-select

- Фильтр по kind (chips: Все / Силовые / Разминка / Растяжка) на странице каталога
  и в `exercise-select` при добавлении в программу.
- Поиск и фильтр по мышцам работают как раньше, комбинируются с kind.

## 3. Редактор программы

- Разминка/растяжка добавляются существующим exercise-select обычным элементом списка;
  позиция — существующий drag&drop (разминка первой, растяжка последней — руками).
- Для `kind !== 'strength'` вместо полей «подходы × повторы × вес» — одно поле
  «длительность» (`target_duration_seconds`), ввод в минутах:секундах или секундах.

## 4. Воркаут

- Для warmup/stretching: один «подход» с вводом времени вместо веса×повторов,
  запись в `workout_log_sets.duration_seconds`, отметка «сделано» как у обычного сета.
- Без таймера — просто ввод значения (таймер — потом, если понадобится).

## 5. Seed-миграция

- ~15–20 записей на русском: суставная разминка (шея, плечи, корпус, таз),
  кардио-разогрев (скакалка, велотренажёр), растяжки по группам мышц.
- `kind` соответствующий, `primary_muscles` — где осмысленно, видео нет.

## 6. Подсказка растяжки в воркауте

- В конце тренировки блок «Растяжка»: собираем primary_muscles упражнений тренировки,
  фильтруем каталог по `kind === 'stretching'` и пересечению мышц,
  сортируем по покрытию, показываем топ-5.
- Тап — добавляет растяжку в текущую тренировку обычным элементом.
- Считаем по запланированным упражнениям, а не по выполненным подходам —
  проще и список не скачет по ходу тренировки.

## Вне скоупа

- Таймер для выполнения разминки/растяжки.
- Автоматическая вставка разминки в программы.
- Подсказки по фактически выполненным подходам.
