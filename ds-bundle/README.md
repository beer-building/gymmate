# GymMate — design foundation

«Терминал тренировок»: интерфейс выглядит как точный прибор, а не фитнес-журнал. Почти чёрные
поверхности, моноширинные подписи капсом, **монохромный** акцент-чернила как голос действия и
прогресса. Цвет на экране зарезервирован за **данными** (мышцы, ошибка, нагрузка), не за брендом.
Цифры (веса, повторы) — герой каждого экрана.

Это **CSS-фундамент**, не библиотека компонентов. Стройте разметку обычными элементами с этими
классами — стилизует их `styles.css`. Никакого провайдера/обёртки не нужно: тема следует
`prefers-color-scheme` сама (тёмная — основная). Подключите `styles.css` — и всё работает.

## Токены (CSS-переменные)

- **Цвет:** `--color-background`, `--color-text`, `--color-accent` (монохромный, = тексту),
  `--color-error`, `--color-surface` (приподнятые плиты), `--color-sunken` (поля ввода),
  `--color-border` / `--color-border-strong`, `--color-muted` (вторичный текст),
  `--color-accent-dim` (hover-границы). Данные: `--color-muscle-primary` (красный),
  `--color-muscle-secondary` (жёлтый).
- **Шрифт:** `--font-display` (Unbounded — только h1–h3), `--font-body` (Manrope — текст),
  `--font-mono` (JetBrains Mono — лейблы, кнопки, **все цифры**).
- **Радиус:** `--radius-sm` (6px), `--radius-md` (8px), `--radius-lg` (12px), `--radius-pill`.
- **Анимация:** `--spring-transition` — пружинящая кривая для откликов.
- Полупрозрачности — через `oklch(from var(--token) l c h / α)`, **не** новыми hex.

## Классы

- **Кнопки:** `.btn` (primary, акцентная заливка) · `.btn.ghost` · `.btn.danger` ·
  размеры `.btn.sm` / `.btn.lg` · `.btn.icon` / `.btn.icon-filled` (квадрат 2rem, цвет через
  `--icon-color`). Mono, UPPERCASE, скругление `sm`.
- **Чипы:** `.chip` (фильтры каталога) · `.chip.active` (акцентная пилюля).
- **Плиты/карточки:** `.plate` · `.plate.glow` (hover-подъём + glow) ·
  `.plate.volt` (заряженная — ключевой блок страницы).
- **Поля:** обёртка `.field` (label + input/select/textarea внутри, фон `sunken`,
  фокус — акцентная граница) · `.error-text` (текст ошибки).
- **Текст:** `.eyebrow` (mono-надпись над заголовком), `.muted`, `.mono`.
- **Лейаут:** `.container` (центр, max 1100px) · `.rise` (анимация появления).

## Правила

- **Монохромный акцент.** Действие/состояние/прогресс маркируются тоном (заливка/инверсия),
  не оттенком. На акцентной заливке текст — цвета фона (`--color-background`).
- **Цвет = данные.** Цветной — только мышцы / ошибка / нагрузка. Иначе экран нейтральный.
- **Табло.** Сравниваемые цифры — `--font-mono` (выравнивание разрядов = читаемость прогресса).
- **Прибор.** `--font-display` запрещён в кнопках, лейблах, данных — только заголовки.
- **Плоско в покое.** Тень/glow — только ответ на состояние (hover, активность), не декорация.
- **Анти-референсы:** глянцевые фитнес-аппы (сток-фото, баннеры, слоганы), корпоративный
  SaaS-дашборд (серые сетки, hero-метрики), геймификация (бейджи, стрики, конфетти),
  медицинский трекер (стерильный белый, синие акценты).

## Пример

```html
<article class="plate glow" style="padding:1.25rem;display:flex;flex-direction:column;gap:.75rem">
  <span class="eyebrow">Следующая тренировка</span>
  <h3>Жим лёжа</h3>
  <p class="mono" style="font-size:1.5rem">80<span class="muted">кг</span> × 8</p>
  <div style="display:flex;gap:.5rem">
    <button class="btn">Записать подход</button>
    <button class="btn ghost">Пропустить</button>
  </div>
</article>
```
